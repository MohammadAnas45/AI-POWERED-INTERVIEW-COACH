# AI-Powered Interview Coach

This is a React + Vite + Tailwind CSS website for the AI-Powered Interview Coach project (Week 1).

## Getting Started

Since this project was generated manually without a Node environment available:

1.  **Install Node.js**: Ensure you have Node.js installed (v18+ recommended).
2.  **Install Dependencies**:
    ```bash
    npm install
    ```
3.  **Run Development Server**:
    ```bash
    npm run dev
    ```

## Project Structure

-   `src/components`: Reusable UI components (Navbar, Hero, Section, Footer)
-   `src/pages`: Page layouts (Home)
-   `src/App.jsx`: Main application component
-   `src/main.jsx`: Entry point

## Tech Stack

-   **Framework**: React (Vite)
-   **Styling**: Tailwind CSS
-   **Icons**: Lucide React
