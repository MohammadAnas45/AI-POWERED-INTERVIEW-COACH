## 🚀 How to Run Locally

```bash
# Clone the repository
git clone https://github.com/your-username/ai-interview-coach.git

# Navigate into project
cd ai-interview-coach

# Install dependencies
npm install

# Start development server
npm run dev
```

App runs on:

```
http://localhost:5173
```
